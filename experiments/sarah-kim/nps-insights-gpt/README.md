# NPS Insights GPT

ChatGPT Custom GPT for NPS analysis.

## Capabilities

- Theme extraction from verbatim responses
- Sentiment trend analysis
- Segment comparison
- Executive summary generation

## Usage

Upload NPS export CSV, ask questions like:
- "What are the top 5 themes from detractors?"
- "Compare Enterprise vs SMB sentiment"
- "Generate executive summary for Q4"
